# Source Code And Simulation Files

Use this folder for project code, scripts, notebooks, Ansys Lumerical simulation files, models, or analysis tools.

Suggested layout once the project direction is confirmed:

```text
src/
|-- analysis/
|-- lumerical/
|-- models/
|-- notebooks/
|-- scripts/
`-- tests/
```

Keep large generated simulation outputs out of Git unless they are small, meaningful, and safe to share. Store raw or bulky outputs in an ignored folder such as `results/raw/`.
