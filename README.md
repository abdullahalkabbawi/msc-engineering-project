# Terahertz Electronics For Quantum Engineering And 6G Communications

This repository documents my MSc Engineering Project for the University of Glasgow course **ENG5059P MSc Project**.

The project investigates terahertz electronics for future quantum engineering and 6G communication technologies, using **Ansys Lumerical** as the main simulation and modelling environment.

This repository is being developed as both an academic project workspace and an engineering portfolio piece. It will include the project plan, technical approach, simulation workflow, results, and final communication materials.

## Project Status

Current stage: project setup, background research, and simulation planning.

## Project Aim

The aim of this project is to explore how terahertz-frequency electronic and photonic structures can support emerging applications in quantum engineering and 6G communications. The work will use Ansys Lumerical to model, simulate, and evaluate relevant device or system behaviour at terahertz frequencies.

The final output will combine a technical report, poster presentation, and a documented simulation workflow that shows the engineering reasoning, design choices, and evaluation results.

## Technical Focus

- Terahertz electronics and photonics
- Quantum engineering applications
- 6G communication technologies
- Electromagnetic or photonic simulation using Ansys Lumerical
